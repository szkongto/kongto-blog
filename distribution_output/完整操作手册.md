# KONGTO 内容分发 - 完整操作手册

## 📋 系统说明

本系统为**半自动化内容分发助手**，可以：
- ✅ 自动将博客文章转换为各平台格式
- ✅ 生成发布清单和操作建议
- ✅ 批量处理所有文章
- ❌ 不能自动登录和发布（平台限制）

---

## 🎯 推荐发布平台（已生成内容）

### 1. 知乎 （已生成问答格式）
**优势**：专业受众、长尾流量
**策略**：
- 搜索相关问题，用生成的内容回答
- 或自问自答（注意社区规范）
- 在个人简介留博客链接

**文件位置**：`distribution_output/知乎/`
- `问题_xxx.txt` - 要提的问题
- `回答_xxx.txt` - 要发布的回答内容

### 2. CSDN （已生成博客格式）
**优势**：专业技术社区、SEO友好
**策略**：
- 直接复制生成的内容发布
- 选择合适的分类和标签
- 首发推荐在这里

**文件位置**：`distribution_output/CSDN/`
- `xxx.md` - 完整博客文章

### 3. 简书 （已生成文章格式）
**优势**：内容创作平台、门槛低
**策略**：
- 内容要稍微修改，避免完全重复
- 适合养号和积累粉丝

**文件位置**：`distribution_output/简书/`

### 4. 百度贴吧 （已生成帖子格式）
**优势**：精准受众（机床吧、数控吧等）
**风险**：容易被删帖、封号
**策略**：
- 以技术交流口吻
- 不要硬广
- 可以回复相关帖子

**文件位置**：`distribution_output/贴吧/`

## 🚀 批量处理剩余文章

测试成功后，运行以下命令批量处理所有文章：

```bash

```

## 🤖 如果需要更高程度的自动化

### 方案A：浏览器自动化（高级）
使用Selenium或Playwright模拟人工操作：

```python
# 示例：使用Playwright自动发布到CSDN
from playwright.sync_api import sync_playwright

def auto_post_csdn(title, content, tags):
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=False)
        page = browser.new_page()
        page.goto('https://mp.csdn.net/')
        # 需要手动登录一次，之后可以保存cookies
        # 然后自动填写表单发布
```

**风险**：
- 仍需处理验证码
- 可能被检测为机器人
- 建议只用于部分流程自动化

### 方案B：使用官方API（如果可用）
部分平台提供API：
- CSDN：有开放API（需申请）
- 知乎：Draft API（限制很多）
- 贴吧：无官方API

### 方案C：第三方分发工具
- **简媒**、**新媒体管家** 等工具
- 支持多平台一键分发
- 但需要付费，且支持的平台有限
